How these are built

  • Tables + inline styles only — what Gmail, Apple Mail, Outlook and iOS Mail actually render.
  • Fraunces, Manrope and Caveat are linked on these preview pages so you can see the intended look, but the signature HTML uses safe fallbacks: Georgia italic for Fraunces, Brush Script / Snell Roundhand / cursive for Caveat, Arial / system-ui for Manrope.
  • Offset “stamp” shadows render in Gmail web, Apple Mail and iOS; Outlook desktop falls back to a clean bordered card.
  • Logo pulls from addedpixels.com/images/logos/addedpixels_logo_standard_dark.png. Avatar and icons reuse the URLs from your current sig.

Installing in Gmail

  • Open your chosen variant page in a browser.
  • Select everything inside the white card (from name to legal footer) and copy.
  • Gmail → Settings → See all settings → Signature → paste. Save.
  • If you prefer pasting raw HTML, expand “Copy raw signature HTML” at the bottom of each variant page and grab the source there.
